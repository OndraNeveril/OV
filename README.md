# OV

Soubory ze stáže Otevřené vědy Akademie věd ČR.

Soubory:

  - get_data.py: Kód na automatické stahování a ukládání dat z databáze Merra 2. Nejprve je potřeba na stránkách vybrat subset a získat seznam souborů, ten potom bere tento kód.

  - graphs.py: Kód, vytvářející grafy rozdílu mezi Merrou a Jawarou za účelem posouzení použitelnosti re-analýzy Jawara. Grafy jsou uložené ve složce graphs.

  - graphs_lat.py: Kód, vytvářející grafy průběhu teploty a zonálního větru v jednotlivých zeměpisných šířkách. Grafy jsou uložené ve složce graphs_lat.

  - zpracovani.py: Soubor s dodatečnými funkcemi usnadňující práci s jednotlivými re-analýzami, umožňující převedení dat do numpy array, z nichž už lze snadno vytvořit graf.
